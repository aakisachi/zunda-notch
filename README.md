# ずんだノッチ（ZundaNotch）

macOS のノッチ（画面上部の切り欠き）まわりに常駐して、**Claude Code** と **Codex** の作業状況を見守る個人用ユーティリティなのだ。作業の完了・承認待ちを **ずんだもんの音声** で知らせて、セッション一覧からワンクリックでその作業画面へ戻れるのだ。

> 個人の学習目的で作った非公式ツールです。Anthropic / OpenAI とは無関係で、公式サポートもありません。

## 主な機能

- **セッション一覧**：稼働中の Claude Code / Codex セッションを、サイドバーの名前つきで表示
- **ずんだもん音声通知**：完了「できたのだ！」／承認待ち「許可がほしいのだ！」（ON/OFF・サイレント時間帯対応）
- **ノッチから承認**：Claude Code の許可リクエストに [許可]/[拒否] ボタンで応答（無応答時は通常のターミナル確認へ自動フォールバック）
- **ピンポイントジャンプ**：セッション行クリックで、その端末タブ（Terminal.app / iTerm2 は tty 精度）や Claude デスクトップの当該セッションへ移動
- **使用量ゲージ**：Claude / Codex の使用率を表示（設定で非表示可）
- **ピクセルバディ**：折りたたみ時にノッチ横で待機する枝豆キャラ

## 動作環境

- macOS 14 (Sonoma) 以降
- Swift 5.9 以降（ビルド時。Xcode は不要）
- 音声通知に [VOICEVOX](https://voicevox.hiroshiba.jp/) （ずんだもん）

## ビルドと導入

```bash
swift build -c release
# 生成物を .app に組み込む例:
cp .build/release/ZundaNotch dist/ずんだノッチ.app/Contents/MacOS/ZundaNotch
codesign --force -s - dist/ずんだノッチ.app
cp -R dist/ずんだノッチ.app /Applications/
open -a "ずんだノッチ"
```

## 必要な権限

- **アクセシビリティ**：セッション行クリックで Claude デスクトップの該当セッションを開くために使用（サイドバーの同名行をアクセシビリティAPIで押下）。初回ジャンプ時に許可を求めます。*システム設定 → プライバシーとセキュリティ → アクセシビリティ* で有効化してください。
- **オートメーション**：Terminal.app / iTerm2 のタブ精度ジャンプに使用（初回に確認ダイアログ）。

## 仕組み（概要）

- Claude Code の hooks と Codex の notify から、ローカルの HTTP サーバ（`127.0.0.1:48765`）にイベントを受け取ります。外部送信はありません。
- サーバは loopback バインドのみ。正規の呼び出し元以外（ブラウザ由来の `Origin`/`Sec-Fetch` 付きリクエスト）は拒否します。
- Claude 公式使用量を使う場合のみ、`claude setup-token` で発行したトークンを **macOS Keychain** に保存し、`api.anthropic.com` への問い合わせにだけ使います（画面・ログ・ディスクには出しません）。

## プライバシー

セッションのタイトルや状態はローカルでのみ扱い、当アプリからクラウドへ送信することはありません（唯一の外部通信は、任意設定した公式使用量トークンによる Anthropic 公式APIへの問い合わせのみ）。

## クレジット

- 音声：VOICEVOX（ずんだもん）— 利用の際は各ソフトウェア/キャラクターの利用規約に従ってください。
